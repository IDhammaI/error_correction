# 错题本生成流程

> 目标：从试卷/作业图片或 PDF 中提取「干净的题目文本（题干/选项/结构）」，并生成可整理的错题本；
> 错题判定：由用户勾选；
> 手写和印刷体分类：使用 Handwritten-and-Machine-Printed-Text-Detection。

## 输入 / 输出

### 输入
- 单张或多张图片（拍卷子/作业）
- PDF（扫描版或拍照版转 PDF）

### 输出
- `questions.json`：每页/每题的结构化结果
- `preview.html` 或 `preview.json`：供用户勾选错题的预览数据
- `wrongbook.md`：最终错题本（任选一种先做）

---

## 1. 文档解码与预处理

### 1.1 PDF 渲染
- PDF -> 按页渲染成图片
- 产物：`page_i.png`

### 1.2 图像预处理
- 裁剪边缘
- 去阴影、增强对比
- 降噪
- 产物：`page_i_pre.png`

---

## 2. 文本检测与裁剪

### 2.1 文本框检测
- 输入：`page_i_pre.png`
- 输出：`page_i_boxes.json`（bbox 坐标 + 置信度）

### 2.2 按 bbox 裁剪文本块
- 输入：`page_i_pre.png` + `page_i_boxes.json`
- 输出：`page_i_crops/box_0001.png ...`

---

## 3. 手写 vs 印刷体分类

> 使用 Handwritten-and-Machine-Printed-Text-Detection 对每个 crop 分类。

- 输入：`page_i_crops/*.png`   （$\text{block\_label}\neq\text{image}$）
- 输出：`page_i_box_labels.json`
  - 每个 bbox：`label` $\in$ `{handwritten, printed}` + `score`

分组得到：
- `printed_boxes`
- `handwritten_boxes`

---

## 4. 生成无手写内容的图片

### 4.1 生成手写 mask
- 输入：`handwritten_boxes`
- 输出：`page_i_hw_mask.png`

### 4.2 应用 mask 得到干净图
- 对手写区域做：涂白 
- 输入：`page_i_pre.png` + `page_i_hw_mask.png`
- 输出：`page_i_clean.png`

## 5. PaddleOCR

### 5.1 调用文档解析API
- 输入：`page_i_clean.png`
- 输出：`page_i_struct.json`
  - `parsing_res_list`：结构化块（含 `block_label / block_content / block_bbox`）
  - `block_order`：阅读顺序
  - `markdown.text`：可直接用的 Markdown
  - `markdown.images`：图片占位到真实 URL 的映射

---

## 6. 题目切分与结构化

### 6.1 阅读顺序
- 按 `parsing_res_list[].block_order` 排序得到 `ordered_blocks`

### 6.2 题目切分
- 在 `ordered_blocks` 中做题号检测：`1.`、`(1)`、`【1】`、`一、` 等
- 划分题目边界
- 输出：`question_spans`

### 6.3 题干与选项装配
- 拼接 `question_spans` 内的 `block_content` 得到 `stem`
- 识别并装配选项：A/B/C/D 或 ①②③④（可为空）
- 处理：
  - 公式块：直接使用 `block_content`
  - 图片块：根据 `markdown.images` 下载到 `assets/` 并在题干中引用
- 输出：`questions.json`

---

## 7. 手写答案留存
- 输入：`page_i_pre.png` + `handwritten_boxes`
- 输出：`handwriting.png`

---

## 8. 用户勾选错题
- 生成预览：基于 `markdown.text` 快速预览
- 用户勾选：`selected_wrong_qids`

---

## 9. 导出错题本
- 输入：`questions.json` + `selected_wrong_qids`
- 输出：`wrongbook.md`（每题：stem + 公式/图片 + 手写截图）

## 10.Tools

1) `prepare_input(path) -> pages[]`
- 图片读取，返回每页图片路径

2) `preprocess_image(page_path) -> page_pre_path`
- 裁边/去阴影/增强/降噪

3) `detect_text_boxes(page_pre_path) -> boxes.json`
- PaddleOCR 检测（仅 bbox）

4) `crop_boxes(page_pre_path, boxes.json) -> crops_dir`
- 按 bbox 裁剪文本块

5) `classify_hw_printed(crops_dir) -> box_labels.json`
- Handwritten-and-Machine-Printed-Text-Detection 分类（handwritten/printed）

6) `make_hw_mask(boxes.json, box_labels.json) -> hw_mask.png`
- 将 handwritten 框生成 mask

7) `apply_mask(page_pre_path, hw_mask.png) -> page_clean.png`
- 涂白

8) `paddleocr_struct(page_clean.png) -> page_struct.json`
- 调 PaddleOCR 文档解析 API（返回 parsing_res_list/block_order/markdown 等）

9) `split_questions(page_struct.json) -> questions_page.json`
- 题号检测、题干/选项装配（输出本页题目结构）

10) `attach_handwriting_assets(page_pre_path, handwritten_boxes) -> handwriting_assets[]`
- 裁剪手写区域并关联到最近题目（MVP 只保存截图）

11) `build_preview(questions.json) -> preview.html/json`
- 生成可勾选预览数据

12) `export_wrongbook(questions.json, selected_wrong_qids) -> wrongbook.md`
- 导出错题本
