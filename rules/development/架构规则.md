# 架构规则

本项目是错题本与笔记整理系统，核心流程包括文件上传、OCR、题目分割、纠错、笔记整理、错题库保存、复习和 AI 对话。

## 代码入口

- `backend/web_app.py`：Flask 应用工厂和 Blueprint 注册。
- `backend/routes/`：后端 API 路由实现。
- `backend/core/`：配置、状态、额度、模型选择、邮件等基础能力。
- `backend/db/`：SQLAlchemy ORM、迁移和 CRUD。
- `backend/agents/`：题目分割、纠错、解题、讲解和笔记整理 Agent。
- `backend/src/workflow.py`：OCR 到题目处理的主工作流。
- `frontend/src/router/index.js`：前端路由来源。
- `frontend/src/api/`：前端 API 调用来源。
- `frontend/src/views/`：页面级组件。
- `frontend/src/components/`：复用组件。
- `frontend/src/composables/`：组合式状态和业务流程。

接口、路由和页面清单以源码为准，不在规则文档中手写完整列表。
