# 环境配置规则

## 前置要求

- 已安装 Node.js 和 npm。
- 已安装 Python 依赖，或已准备好后端虚拟环境。
- 后端依赖使用 `requirements.txt`。
- 前端依赖使用 `frontend/package.json`。

安装前端依赖：

```bash
cd frontend
npm install
```

## 环境变量

- `.env` 放在项目根目录，不提交到仓库。
- `backend/.env.example` 是环境变量模板。
- 必须配置 `SECRET_KEY`。
- Flask 级配置、SMTP 邮件配置和 LangSmith 配置通过 `.env` 管理。
- LLM / OCR Provider 配置存储在数据库中，由系统设置页面管理。
- 新增环境变量时必须同步更新 `backend/.env.example`。
- 涉及环境变量或部署配置变更时，必须同步说明默认值、必填项和兼容影响。
